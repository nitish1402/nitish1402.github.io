Computing C(M,N)%MOD

July 7, 2014 by Nitish

This blog post is about calculating C(M,N)%MOD where MOD=100000007 and N and M are given numbers

Calculating C(M, N)

Since we have to calculate the answer modulo MOD = 1,000,000,007, the above formula have to be rewritten to

C(M, N) = (M)! × N!-1 × (M-N)!-1) mod MOD

Here, x-1 is the modular multiplicative inverse of x modulo MOD. Since MOD is a prime number, we can calculate it using Fermat's little theorem: x-1 = xMOD-2 (mod MOD). We can calculate xMOD-2in O(lg MOD) time using exponentiation by squaring. Because we will use factorials many times in this solution, we can precompute all factorials and their inverses (modulo MOD) for all integers from 0 through (let)800,000, inclusive.

                
                  void preProcess()
                  {
                      fact[0]=1;
                      ifact[0]=1;
                      int i;
                      for(i=1;i1;i--)
                        ifact[i-1] = ifact[i]*i%MOD;
                  }

Therefore the function comb(X, Y) can be calculated in constant time as follows. For convenience, we also introduce function ways(N, M) .

                
                  ll comb(int a,int b)
                  {
                     return (((fact[a]*ifact[b])%MOD)*ifact[a-b])%MOD;
                  }
